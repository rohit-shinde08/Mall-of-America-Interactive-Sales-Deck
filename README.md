# Mall-of-America-Interactive-Sales-Deck
 Mall of America Interactive Sales Deck is an AI-powered immersive presentation designed to showcase one of America’s largest shopping and entertainment destinations. The project combines cinematic storytelling, luxury-inspired UI/UX, interactive animations, and responsive frontend development to create a premium digital experience. It highlights retail opportunities, entertainment attractions, sponsorship potential, and event hosting capabilities through visually engaging and data-driven content. Built with modern web technologies, the platform delivers smooth performance, non-linear navigation, and video-first storytelling for prospective tenants, sponsors, and event partners. The experience is inspired by luxury brands and modern interactive digidecks to create a bold, high-impact presentation.
